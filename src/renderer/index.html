Преобразование значения

Доступные переменные и функции:
value — текущее сгенерированное значение поля
row — объект всей строки (row["цена"])
col(name) — безопасное получение значения столбца
IF(условие, да, нет) — условная логика (вложенная поддерживается)
Примеры:
IF(value > 100, "Высокий", "Низкий")
row["скидка"] ? value * 0.9 : value
